### Install

```sh
git clone https://github.com/peguimasid/42-Libasm.git libasm
```

### Usage

1. Ensure you are in a x86_64 environment (`uname -m` should return `x86_64`).
2. Navigate to the project directory:
  ```sh
  cd libasm
  ```
3. Build and run tests:
  ```sh
  make test
  ```
